About KCR

Overview

Name: KCR (Kenzo's Computer Research)
Founder: Kenzo Basar
Type: Software Development Group
Founded: 2024
Succeded by: J8gho Group

KCR (Kenzo's Computer Research) is a software development group that focuses on creating lightweight, efficient tools and utilities for various platforms. Founded by Kenzo Basar, the group is known for its minimalistic approach to development, creating software solutions that prioritize speed and low system resource usage.

Notable Projects

  • LKT (Lightweight Text Editor)
  • OSbbin
  • Kenzo OS

LKT (Lightweight Text Editor)

LKT is a lightweight text editor designed by KCR for users who want a fast, simple, and minimalistic tool for basic text editing. It is only 10 KB in size, making it ideal for low-storage systems.

OSbbin

OSbbin is an operating system shell that includes various utilities like a sticky notes widget, web browser, and clock. It aims to provide an efficient and basic environment for users who prefer simplicity.

Kenzo OS

Kenzo OS is a custom operating system developed by Kenzo Corporation, not from KCR but from the family's oldest group.

Tools Developed

  • KCR-SOFT: A suite of development tools and utilities for various purposes.
  • KenzoCode: An efficient code editor with features like syntax highlighting, multi-language support, and integrated web browsing.
  • Lin8 Emulator: A tool to emulate operating system environments for development and testing.
